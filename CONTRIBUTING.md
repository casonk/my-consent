# Contributing

This repository holds personal consent and data-processing consent statements.

## Workflow

1. Add or update consent documents in the repo root or a relevant subdirectory.
2. Update `README.md` to reflect any new document.
3. Use a focused Conventional Commit such as `docs: add consent statement for <service>`.

## Content Standards

- Preserve the intent and wording of existing consent documents unless substantive revision is explicitly needed.
- Do not commit private personal identifiers or contact details beyond what the consent text is meant to disclose publicly.
- Prefer additive, clearly explained documentation updates over rewrites.
- Avoid committing operational tool configuration or local environment files.

## Pull Requests

All contributions should go through a pull request with a clear description of the change and its purpose.
