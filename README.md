# my-consent

This repository hosts my personal consent statements for specific services and for my own portfolio of personal tools that process my personal data.

## Documents

- `personal-tools.md` — general consent for my personally owned or operated tools and repositories to process my personal data for their intended functions
- `ai-and-model-processing.md` — consent for local or hosted model-assisted extraction, classification, summarization, drafting, and agentic workflows
- `financial-records.md` — consent for personal financial statements, balances, receipts, reporting, and related automation
- `health-and-supplement-data.md` — consent for personal supplement, ingredient, dosage, sourcing, and comparison workflows
- `messaging-and-email.md` — consent for personal messaging, email, notification, and inbox-processing workflows
- `credentials-and-secrets.md` — consent for personal credential, secret, and secure-store automation
- `remote-access-and-private-files.md` — consent for personal file-sharing, remote-access, device-profile, and access-control workflows
- `twilio.md` — Twilio toll-free messaging consent
